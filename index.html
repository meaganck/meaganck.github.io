Grade Calculator

Java javafx


Add your grades to this GUI, and it will show your current grade before an exam and what mark you need on your exam to achieve your goal! This program uses the javafx library to create the user interface window. I also used the Model-View-Controller design to structure this program. The model contains the user's grades that they input, and methods that sort and calculate their grade. The view contains all the code for how the GUI looks, and there is a class for each window(the main window, and classes for each pop-up window). The controller handles all the user interactions with the buttons from the view. The list can be sorted by name, grade or weight.
